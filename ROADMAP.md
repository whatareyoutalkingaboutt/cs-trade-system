## 🚀 快速开始

**从阶段0开始：**

```bash
# 1. 进入项目目录
cd ~/cs-item-scraper

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 复制环境变量配置
cp .env.example .env

# 4. 编辑 .env 文件，配置API Key
nano .env

# 5. 运行验证脚本
python scripts/validation/01_test_historical_api.py
python scripts/validation/02_test_realtime_scraper.py

# 6. 验证成功后，开始阶段1开发
```
